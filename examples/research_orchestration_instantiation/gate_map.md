# Condensed Gate Map

This condensed map keeps the public example readable while showing the general
shape of governed progression.

- `Gate0`
  - question clarity and project bootstrap readiness
- `Gate1`
  - source inventory and pre-protocol evidence sufficiency
- `Gate2`
  - protocol lock completeness and redesign-boundary clarity
- `Gate3`
  - knowledge build sufficiency for serious downstream work
- `Gate4`
  - analysis dataset rebuildability and QA readiness
- `Gate5`
  - analysis reproducibility and deviation visibility
- `Gate6`
  - publishability and route-worthiness review
- `Gate6A`
  - methodological audit completeness before writing
- `Gate7`
  - manuscript drafting completeness and claim traceability
- `Gate8`
  - citation bundle coherence
- `Gate9`
  - journal-fit decision completeness
- `Gate10`
  - submission package completeness
- `Gate11`
  - final QA and explicit human-approval readiness

The example coordinate `r_protocol_lock` is most closely associated with
`Gate2`.
